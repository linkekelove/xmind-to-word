# Changelog

Changelog for this project, following the [Keep a Changelog](https://keepachangelog.com/en/1.1.0/) format.

## [1.1.0] - 2026-07-23

### Fixed

- 修复正文使用两端对齐导致短句文字被强行拉伸、字间距过大分家的问题，改为左对齐
- 清理从设备日志 / 诊断工具复制粘贴带来的全角空格、不间断空格、零宽字符等不可见排版污染字符
- 显式重置字符间距为标准值，避免继承样式或外部来源写入的加宽磅值

### Added

- 新增标题 4 级别，与标题 1~3 一样使用黑体加粗区分层级（字号与正文一致）
- 正文段落新增段后间距，避免大段文字紧贴堆叠导致可读性下降
- 新增标题层级自动编号：二级标题用中文数字（一、二、三），三级 / 四级标题用阿拉伯数字多级编号（1.1、1.1.1）；一级标题若为文档唯一根节点，则自动跳过编号

- 应用中文排版格式：黑体标题、宋体正文、首行缩进 2 字符等
